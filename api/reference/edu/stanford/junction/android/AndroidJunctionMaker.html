public static void castActivity (Context context, ActivityScript script, Cast support)

Sends an intent to complete the casting of an activity. The result will be an Intent issued, of action ACTION_JOIN. The Intent will be populated with casting information, and should be handled in your android.app.Activity: if AndroidJunctionMaker.isJoinable(this) { maker.newJunction(this,mActor); }

public static void castOrJoinActivity (Context context, ActivityScript script, Cast support)

Casts or joins an activity, as decided by the Activity Director. If the Director detects the user is launching to a generic target (Eg a director's activity) then it will start a new activity. If the user pairs with an existing activity, then the user is joined to that session with no casting done.
